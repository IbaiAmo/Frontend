-A-

Algoritmo
Conjunto de reglas bien definidas para la resolución de un problema. Un programa de software es la transcripción, en lenguaje de programación, de un algoritmo.

ASCII
Conjunto de caracteres, letras y símbolos utilizados en todos los sistemas de computadoras de cualquier país e idioma. Permite una base común de comunicación. Incluye a las letras normales de alfabeto español, con excepción de la ñ y toda letra acentuada.

AVI
Formato de Microsoft para archivos de audio y video.

Antivirus
Programa que busca y eventualmente elimina losn virus informáticos que pueden haber infectado un disco rígido, disquete, pendrive, memoria usb, etc...

-B-

Bit
La unidad más pequeña de información en un sistema binario. Puede ser 0 o 1 y se utiliza en la representación de datos en computadoras.

Backup
Copia de seguridad de los datos y archivos importantes para prevenir la pérdida de información en caso de fallos del sistema, errores humanos o eventos catastróficos.

Banda Ancha
Un tipo de conexión a Internet de alta velocidad que permite la transferencia rápida de datos, adecuada para streaming, juegos en línea y descargas de archivos grandes.

Bluetooth
Una tecnología de comunicación inalámbrica que permite la conexión de dispositivos, como auriculares, teclados y altavoces, a través de ondas de radio de corto alcance.

-C-

Caché
Un espacio de almacenamiento temporal en la memoria que almacena datos comúnmente utilizados, acelerando el acceso y la ejecución de programas.

Código Fuente
El conjunto de instrucciones legibles por humanos escritas en lenguaje de programación que los programadores utilizan para desarrollar software.

Criptografía
La práctica de asegurar la comunicación y los datos mediante el uso de técnicas matemáticas y algoritmos para cifrar y descifrar información.

Cortafuegos (Firewall)
Un sistema de seguridad de red que protege una red o computadora al controlar y filtrar el tráfico de datos entrante y saliente.

-D-

DNS (Sistema de Nombres de Dominio)
Un sistema que traduce los nombres de dominio (como www.ejemplo.com) en direcciones IP, permitiendo a los navegadores encontrar servidores en la web.

Debugger
Una herramienta de software utilizada por programadores para identificar y corregir errores en el código fuente de un programa.

Disco Duro (Hard Drive)
Un dispositivo de almacenamiento de datos no volátil que almacena información en un disco magnético giratorio, comúnmente utilizado en computadoras.

Descargar (Download)
El proceso de transferir datos desde un servidor remoto a un dispositivo local, como la descarga de archivos desde Internet.

-E-

Encriptación (Encryption)
El proceso de convertir datos legibles en una forma ilegible (cifrada) para proteger la privacidad y la seguridad de la información durante su transmisión y almacenamiento.

Ethernet
Un estándar de red por cable comúnmente utilizado que permite la comunicación entre dispositivos en una red local.

Expresión Regular (Regex)
Una secuencia de caracteres que define un patrón de búsqueda, utilizado en programación para buscar y manipular texto de manera eficiente.

E-Mail (Correo Electrónico)
Un método de comunicación que permite el intercambio de mensajes y archivos entre usuarios a través de Internet.

-F-

Firmware
Software integrado en dispositivos electrónicos, como routers, impresoras o cámaras, que controla su funcionamiento básico y hardware.

Freeware
Software que se distribuye de forma gratuita, permitiendo su uso sin costo alguno.

Firewall
Un sistema de seguridad que protege una red o dispositivo controlando el tráfico entrante y saliente y bloqueando amenazas potenciales.

FTP (Protocolo de Transferencia de Archivos)
Un protocolo utilizado para transferir archivos entre computadoras a través de una red, como Internet.

-G-

GPU (Unidad de Procesamiento Gráfico)
Un procesador diseñado para acelerar el procesamiento de gráficos y cálculos intensivos, comúnmente utilizado en juegos y aplicaciones de diseño.

Gigabyte (GB)
Una unidad de medida de almacenamiento que representa aproximadamente mil millones de bytes.

HTML (Lenguaje de Marcado de Hipertexto)
Un lenguaje utilizado para crear páginas web, que define la estructura y el formato de los documentos.

HTTPS (Protocolo de Transferencia de Hipertexto Seguro)
Una versión segura de HTTP que cifra la comunicación entre un navegador web y un servidor, protegiendo la privacidad del usuario.

-H-

Hardware
Los componentes físicos de una computadora o dispositivo electrónico, como procesadores, discos duros, teclados y pantallas.

Host
Una computadora o dispositivo que proporciona servicios, recursos o datos a través de una red, como un servidor web.

HTML5
La última versión del lenguaje de marcado HTML, que incluye nuevas características y capacidades para el desarrollo web.

Hacker
Una persona que explora y manipula sistemas informáticos con habilidades técnicas, ya sea por razones éticas o maliciosas.

-I-

IP (Protocolo de Internet)
Un protocolo de red que permite la comunicación y el enrutamiento de datos a través de Internet y otras redes.

Icono
Un símbolo gráfico o imagen que representa una aplicación, archivo o función en una interfaz de usuario.

Interfaz de Usuario (UI)
La parte de un software o dispositivo que permite a los usuarios interactuar con él, incluyendo botones, menús y elementos visuales.

Inteligencia Artificial (IA)
La simulación de procesos de inteligencia humana por parte de máquinas, que puede incluir aprendizaje automático y reconocimiento de patrones.

-J-

JavaScript
Un lenguaje de programación ampliamente utilizado en el desarrollo web para agregar interactividad y funcionalidad a las páginas web.

Java
Un lenguaje de programación versátil utilizado en una variedad de aplicaciones, desde desarrollo web hasta aplicaciones móviles y sistemas integrados.

JPEG (Joint Photographic Experts Group)
Un formato de archivo de imagen que utiliza compresión para reducir el tamaño de los archivos de imágenes fotográficas.

Joystick
Un dispositivo de entrada que permite a los usuarios controlar la dirección y los movimientos en videojuegos y aplicaciones.

-K-

Keylogger
Un tipo de software malicioso que registra las pulsaciones de teclas de un usuario sin su conocimiento o consentimiento, a menudo utilizado para el robo de contraseñas y datos personales.

Kerning
El ajuste de espaciado entre caracteres en tipografía, utilizado para mejorar la legibilidad y el aspecto del texto impreso o en pantalla.

Kilobyte (KB)
Una unidad de medida de almacenamiento que representa aproximadamente mil bytes.

Kernel
El núcleo de un sistema operativo, que gestiona el acceso al hardware y proporciona servicios esenciales para el funcionamiento de la computadora.

-L-

LAN (Red de Área Local)
Una red de computadoras que cubre un área geográfica limitada, como una casa, una oficina o un campus, y permite la comunicación entre dispositivos locales.

Latencia
El tiempo de retraso en la comunicación de datos a través de una red, generalmente causado por la distancia o la congestión de la red.

Logotipo (Logo)
Un diseño gráfico que representa una empresa, marca o entidad, utilizado para identificación y reconocimiento visual.

Linux
Un sistema operativo de código abierto basado en el kernel de Linux, utilizado en una amplia variedad de aplicaciones, desde servidores hasta dispositivos móviles.

-M-

Malware
Software malicioso diseñado para dañar o infiltrarse en sistemas informáticos, incluyendo virus, troyanos y programas espía.

Megabyte (MB)
Una unidad de medida de almacenamiento que representa aproximadamente un millón de bytes.

Memoria RAM (Random Access Memory)
La memoria de acceso aleatorio que permite a una computadora almacenar y acceder a datos en tiempo real mientras se ejecutan aplicaciones.

Módem
Un dispositivo que permite la comunicación entre una computadora y una red, generalmente a través de líneas telefónicas o conexiones de banda ancha.

-N-

Navegador Web
Una aplicación de software que permite a los usuarios acceder y visualizar páginas web en Internet, como Google Chrome, Mozilla Firefox y Microsoft Edge.

Nube (Cloud)
Un servicio de almacenamiento y procesamiento de datos en servidores remotos a los que se accede a través de Internet, permitiendo la sincronización y el acceso a archivos desde cualquier lugar.

NFC (Comunicación de Campo Cercano)
Una tecnología de comunicación inalámbrica de corto alcance que permite la transferencia de datos entre dispositivos cuando están cerca uno del otro.

Node.js
Un entorno de tiempo de ejecución de JavaScript que permite la creación de aplicaciones de servidor altamente escalables y en tiempo real.

-O-

OCR (Reconocimiento Óptico de Caracteres)
La tecnología que permite a las computadoras leer texto impreso o escrito a mano y convertirlo en texto digital editable.

Overclocking
El proceso de aumentar la velocidad de funcionamiento de componentes de hardware, como CPU o GPU, para mejorar el rendimiento.

Open Source (Código Abierto)
Software cuyo código fuente es accesible y modificable por cualquier persona, lo que promueve la colaboración y la transparencia.

OpenGL
Una API gráfica utilizada para desarrollar aplicaciones y juegos en 2D y 3D en una variedad de plataformas.

-P-

Procesador (CPU)
La unidad central de procesamiento de una computadora, que ejecuta instrucciones y realiza cálculos.

Píxel (Pixel)
La unidad más pequeña de una imagen digital, que contiene información sobre el color y la posición en la pantalla.

Protocolo
Un conjunto de reglas y estándares que rigen la comunicación y el intercambio de datos en una red o sistema.

Programación
El acto de escribir instrucciones y algoritmos en lenguajes de programación para que una computadora realice tareas específicas.

-Q-

QuickTime
Un marco de software desarrollado por Apple para la reproducción de contenido multimedia, incluyendo audio y video.

Quad-Core
Un procesador con cuatro núcleos de procesamiento que permite realizar múltiples tareas de manera eficiente.

Query
Una solicitud de información o búsqueda realizada en una base de datos, un motor de búsqueda o una aplicación.

Quantum Computing (Computación Cuántica)
Un enfoque de la informática que utiliza principios de la mecánica cuántica para realizar cálculos a una velocidad mucho mayor que las computadoras clásicas.

-R-

Router
Un dispositivo de red que dirige el tráfico de datos entre diferentes redes, como la red local y la red de Internet.

RAM (Memoria de Acceso Aleatorio)
La memoria que permite a una computadora almacenar y acceder a datos en tiempo real mientras se ejecutan aplicaciones.

Red (Network)
Un conjunto de dispositivos interconectados que pueden comunicarse y compartir recursos, como Internet o una red local.

Responsive Design (Diseño Responsivo)
El enfoque de diseño web que asegura que un sitio web se adapte y se vea bien en una variedad de dispositivos y tamaños de pantalla.

-S-

SSL (Secure Sockets Layer)
Un protocolo de seguridad que cifra la comunicación entre un navegador web y un servidor, garantizando la privacidad de los datos.

Streaming
La transmisión en tiempo real de contenido multimedia, como audio y video, a través de Internet sin necesidad de descargarlo por completo antes de reproducirlo.

Software
Los programas y aplicaciones informáticas que proporcionan funcionalidad a una computadora o dispositivo, incluyendo sistemas operativos, aplicaciones de productividad y juegos.

Script
Un conjunto de instrucciones o comandos escritos en lenguajes de script que realizan tareas automáticas o manipulan datos.

-T-

Terabyte (TB)
Una unidad de medida de almacenamiento que representa aproximadamente un billón de bytes.

Torrent
Un método de distribución de archivos que permite la descarga de archivos desde múltiples fuentes, a menudo utilizado para compartir archivos grandes.

Tablet
Un dispositivo portátil con pantalla táctil que permite la navegación web, el consumo de medios y la ejecución de aplicaciones.

Touchscreen (Pantalla Táctil)
Una pantalla que permite la interacción táctil con los dedos o un lápiz, comúnmente utilizada en dispositivos móviles y tabletas.

-U-

USB (Universal Serial Bus)
Un estándar de conexión que permite la transferencia de datos y la conexión de dispositivos a través de un puerto USB en computadoras y otros dispositivos.

URL (Localizador Uniforme de Recursos)
Una dirección que especifica la ubicación de un recurso en Internet, como una página web o un archivo.

Usuario
Una persona o entidad que utiliza una computadora, dispositivo o servicio en línea para llevar a cabo tareas y acceder a información.

Unidad de Red
Una ubicación en una red local o en la nube que se asigna como una unidad de almacenamiento accesible a través de una letra de unidad o una dirección UNC.

-V-

VPN (Red Privada Virtual)
Una tecnología que permite establecer conexiones seguras a través de una red pública, protegiendo la privacidad y la seguridad de la comunicación.

Virus
Un tipo de malware que se adjunta a programas o archivos y se propaga de un sistema a otro, causando daños o problemas.

VoIP (Voz sobre IP)
Una tecnología que permite la transmisión de voz y comunicación telefónica a través de Internet, como Skype y WhatsApp.

Visualización de Datos
La representación gráfica de datos para facilitar la comprensión y el análisis, a menudo mediante gráficos y tablas.

Wireframe
Una representación visual básica de la estructura y el diseño de una página web o aplicación, utilizada en la etapa de planificación y diseño.

Worm (Gusano)
Un tipo de malware que se replica a sí mismo y se propaga a través de sistemas y redes sin necesidad de un archivo o programa huésped.

WWW (World Wide Web)
El sistema de información en línea que permite a los usuarios acceder y navegar por documentos y recursos en Internet mediante enlaces hipertexto.

XSS (Cross-Site Scripting)
Una vulnerabilidad de seguridad en aplicaciones web que permite a los atacantes inyectar scripts maliciosos en páginas web visitadas por otros usuarios.

XLR (Conector de audio XLR)
Un tipo de conector utilizado en equipos de audio profesional para conectar micrófonos y otros dispositivos de audio a mezcladores y amplificadores.

Xeon
Una línea de procesadores fabricada por Intel, diseñada para servidores y estaciones de trabajo de alto rendimiento.
