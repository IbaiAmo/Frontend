Definiciones de algunos términos

HTTP

Protocolo de la web para transferir datos

HTML

Lenguaje de marcado para crear páginas web.

URL

Dirección web que identifica recursos

TCP/IP

Conjunto de protocolos para la comunicación en Internet

Internet

Red global de computadoras interconectadas

W3C

Organización que establece estándares web

Mozilla Fundation

Organización sin ánimo de lucro que desarrolla software de código abierto
